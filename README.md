# BAP - Musée du Jouet

Dans le cadre d'une Bourse au projet à l'IIM Digital School, le musée du Jouet de Poissy nous a contactés pour la création d'une webapp. Cette application a pour but de mettre en avant les jouets du musée et de proposer des activités interactives pour les enfants.

## Technologies utilisées 🛠️
- React
- Tailwind CSS

## Installation 🚀
1. Cloner le dépôt
   ```bash
   git clone https://github.com/miicolas/bap-musee-du-jouet.git
   ```
2. Installer les dépendances
  ```bash
  npm install
  ```


Créer un fichier .env.local à la racine du projet et ajouter les variables d'environnement suivantes

3. Lancer le serveur de développement
  ```bash
  npm run dev
  ```
